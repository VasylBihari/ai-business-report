# 📊 Mobile Store Business Analysis (AI-Assisted Report)

This project demonstrates how raw business data can be transformed into a clear, decision-ready executive report.

The goal is not just to analyze data, but to provide **actionable insights** that help business owners understand performance, identify risks, and make better decisions.

---

## 🚀 Project Overview

Using a mobile phone retail dataset, this project builds an end-to-end analysis pipeline:

**Raw Data → KPI Calculation → Structured Insights → Executive Report**

The final output is a **management-ready business report** generated with AI assistance (interpretation only, no calculations).

---

## 📌 Business Questions

- What drives revenue and profit?
- Which products and channels perform best?
- Where is the business losing money?
- How do discounts and returns affect profitability?
- What actions should be taken next?

---

## 📈 Key Insights

- ~31% overall profit margin, with heavy discounting (~58% of sales)
- High return rate (~22.5%), concentrated in specific models
- Online channel outperforms physical stores in both margin and return rate
- Some high-revenue products underperform in profitability

---

## 🧠 Approach

1. **Data Validation**
   - Checked for missing values, anomalies, and inconsistencies

2. **KPI Calculation (Python / pandas)**
   - Revenue, profit, margin, return rate, discount impact
   - Aggregations by brand, model, channel, geography, and time

3. **Structured Data Packaging**
   - KPIs converted into JSON format for AI processing

4. **AI-Assisted Analysis**
   - AI used only for interpretation and report generation
   - No calculations performed by the model

5. **Report Generation**
   - Output: structured Markdown → converted to PDF using Quarto

---

## 🛠 Tools Used

- Python (pandas)
- Excel

- Quarto (Markdown → PDF)
