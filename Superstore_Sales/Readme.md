# AI-Generated Business Performance Report

This project demonstrates an end-to-end workflow for generating an executive-level business performance report using Python and structured sales data (CSV).

## Project Overview
The solution automates data preparation and leverages AI to transform pre-aggregated business metrics into a clear, management-ready narrative report.

## Key Features
- Data cleaning and preparation in Python (pandas)
- Business-focused KPIs and summaries
- AI-generated executive report (Markdown/PDF-ready)
- No manual calculations inside the report logic

## Tech Stack
- Python (pandas)
- Jupyter Notebook
- OpenAI API
-  CSV data sources

## Outcome
The final output is a concise, presentation-ready [business report](https://github.com/VasylBihari/ai-business-report/blob/main/Superstore_Sales/output/business_report%20(1).pdf) suitable for decision-makers, demonstrating how AI can support analytical storytelling.
